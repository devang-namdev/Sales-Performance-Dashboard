## 📌 Project Overview

This project focuses on analyzing retail sales data to identify business trends, monitor key performance indicators (KPIs), and support data-driven decision-making.

The dashboard provides comprehensive insights into sales performance, regional revenue, product category analysis, monthly sales trends, discount impact, and sales representative performance.

The objective of this project is to demonstrate practical Data Analytics skills including data cleaning, KPI development, dashboard design, business intelligence, and storytelling using data.

---

# 🎯 Business Problem

Organizations generate large volumes of sales data every day, but extracting meaningful insights can be challenging.

This project solves that problem by creating an interactive dashboard that helps stakeholders:

- Monitor overall business performance
- Identify top-performing products
- Compare regional sales
- Track monthly revenue trends
- Evaluate sales representative performance
- Support strategic business decisions

---

# 📊 Key Performance Indicators (KPIs)

The dashboard tracks the following KPIs:

- 💰 Total Sales Revenue
- 📦 Total Units Sold
- 🛒 Total Transactions
- 🏷 Average Discount %
- 📈 Monthly Revenue Trend
- 🌍 Regional Sales Performance
- 👕 Product Category Revenue
- 👨‍💼 Sales Representative Performance
